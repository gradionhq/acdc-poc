# Support

## Where to get help

| Channel                                                                 | When to use                                                   |
| ----------------------------------------------------------------------- | ------------------------------------------------------------- |
| [GitHub Issues](https://github.com/gradionhq/acdc-poc/issues)           | Bug reports and concrete feature requests (use the templates) |
| [GitHub Discussions](https://github.com/gradionhq/acdc-poc/discussions) | Questions, ideas, general conversation                        |

## What is in scope

- Questions about setting up and running this template.
- Bug reports against the `main` branch (please search existing issues first).
- Feature requests that fit the project's scope (full-stack notes template + agent-ready OSS hygiene).

## What is out of scope

- General Node.js, React, or TypeScript questions — please consult their respective documentation or communities.
- Support for forked or heavily modified versions of this template.

## Security issues

Do **not** open a public issue for security vulnerabilities. See [SECURITY.md](SECURITY.md) for the private reporting channel.
